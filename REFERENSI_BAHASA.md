# Referensi Bahasa MinangScript

Dokumen ini berisi daftar lengkap pemetaan (*mapping*) kata kunci (keywords) dan fungsi bawaan (built-in functions) dari bahasa Python ke bahasa MinangScript.

## 1. Kata Kunci Dasar (Keywords)

### Boolean & Nilai Khusus
* `True` ➔ `Bana`
* `False` ➔ `Salah`
* `None` ➔ `Kosong`

### Logika
* `and` ➔ `jo`
* `or` ➔ `atau`
* `not` ➔ `indak`

### Percabangan
* `if` ➔ `kok`
* `elif` ➔ `kok_lain`
* `else` ➔ `lainnyo`

### Perulangan
* `for` ➔ `untuak`
* `while` ➔ `salamo`
* `in` ➔ `di`
* `break` ➔ `baranti`
* `continue` ➔ `taruih`
* `pass` ➔ `lewat`

### Fungsi
* `def` ➔ `buek`
* `return` ➔ `baliakan`
* `yield` ➔ `hasilkan`
* `lambda` ➔ `fungsi_ketek`

### Class / Pemrograman Berorientasi Objek (OOP)
* `class` ➔ `kalas`
* `self` ➔ `awak`
* `super` ➔ `induak`

### Penanganan Error (Exception Handling)
* `try` ➔ `cubo`
* `except` ➔ `kacuali`
* `finally` ➔ `akhirnyo`
* `raise` ➔ `angkek`
* `assert` ➔ `pastikan`

### Import & Modul
* `import` ➔ `ambiak`
* `from` ➔ `dari`
* `as` ➔ `sagai`

### Scope / Konteks
* `global` ➔ `sadoalah`
* `nonlocal` ➔ `indak_lokal`
* `with` ➔ `jo_ko`

### Asynchronous (Python 3.5+)
* `async` ➔ `basamo`
* `await` ➔ `tunggu`

### Pattern Matching (Python 3.10+)
* `match` ➔ `cocok`
* `case` ➔ `kasus`

### Lainnya
* `del` ➔ `hapuih`
* `is` ➔ `ialah`
